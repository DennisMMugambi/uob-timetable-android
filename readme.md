# UoB timetable

This application displays the 5-day timetable for any University of Bedfordshire 
course.

![Hero image](screenshots/art/hero.png "Hero image")

## Features

- Detailed information for each session including the unit, time, rooms and 
staff members
- Session hiding
- Term dates
- Offline support
